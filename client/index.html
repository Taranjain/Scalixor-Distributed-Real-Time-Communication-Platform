Everything you need

A complete real-time platform designed for developer workflows.

Real-Time Chat

Instant group rooms with message history, typing indicators, and persistent storage across distributed nodes.

Collaborative Editor

Multi-language syntax highlighting with real-time sync. Perfect for code reviews, interviews, and teaching DSA.

1:1 Video Calling

WebRTC peer-to-peer video calls with mic and camera controls. No media flows through the server.

Horizontally Scalable

Redis Pub/Sub, sticky sessions, and multiple WebSocket instances. Scale out without limits.

Architecture

Built for scale from day one.

   Clients (Browser)
        │
        │  WebSocket (Chat + Signaling)
        ▼
   ┌─── Caddy (Reverse Proxy / Load Balancer) ───┐
   │            ip_hash sticky sessions           │
   ▼            ▼            ▼
 ws1:4001    ws2:4002    ws3:4003    ← Multiple Backend Instances
   │            │            │
   └────────────┼────────────┘
                │  Redis Pub/Sub
                ▼
         ┌──────────────┐
         │  Redis Server │
         │  • Chat Channel
         │  • Signaling Channel
         │  • Online Users Set
         └──────────────┘

After WebRTC signaling completes:
   Client A 🎥 ←─── P2P Media (WebRTC) ───→ 🎥 Client B
